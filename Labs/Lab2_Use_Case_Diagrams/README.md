<h2>1. ATM System</h2><br>
This diagram represents a simple ATM system where a client interacts with the machine to manage funds, and an admin oversees operations. The client can select an amount, withdraw cash, or log out. The system admin monitors cash flow and views client transactions to ensure the system prevents invalid usage.

<h2>2. Medical Clinic</h2><br>
This diagram models a medical clinic's scheduling and billing. Patients initiate the system by signing up, while receptionists schedule appointments and register payments. Doctors make appointments and write prescriptions, with nurses assisting the process.

<h2>3. Car Insurance System</h2><br>
This system illustrates the interactions between car owners and insurers. The car owner insures their car and makes payments, while the insurer searches for the car in the database, adds it if unregistered, accepts the insurance, and confirms payments.
